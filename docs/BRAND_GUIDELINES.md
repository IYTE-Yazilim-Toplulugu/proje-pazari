# IYTE Proje Pazarı Brand Guidelines

## Colors

### Primary Colors
- **IYTE Blue**: `#0ea5e9` - Main brand color
- **IYTE Dark Blue**: `#0284c7` - Hover states, accents
- **IYTE Light Blue**: `#38bdf8` - Backgrounds, highlights

### Usage
- Use IYTE Blue for primary actions (buttons, links)
- Use Dark Blue for hover states
- Use Light Blue for backgrounds and subtle accents

## Typography

### Fonts
- **Primary**: Geist Sans
- **Monospace**: Geist Mono

### Hierarchy
- H1: 3rem (48px), font-bold
- H2: 2.25rem (36px), font-semibold
- Body: 1rem (16px), font-normal

## Logo Usage

### Clear Space
- Maintain minimum clear space around logo (equal to height of "İ")
- Don't place logo on busy backgrounds

### Minimum Sizes
- Digital: 32x32px
- Print: 20mm

### Don'ts
- Don't stretch or distort logo
- Don't change logo colors
- Don't add effects (shadows, gradients)

## Icons

Use IYTE logo icon for:
- Favicons
- App icons
- Social media avatars

Use full logo for:
- Headers
- Marketing materials
- Documentation
